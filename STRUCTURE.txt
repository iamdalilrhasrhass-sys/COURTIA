   $ npm install
   $ cp .env.example .env
   $ npm run dev

3. FRONTEND SETUP:
   $ cd ~/Desktop/CRM-Assurance/frontend
   $ npm install
   $ npm run dev

4. VOIR: http://localhost:3001 (frontend)
          http://localhost:3000 (API)
